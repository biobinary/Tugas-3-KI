Secure Chat

Flow:

  1. User gets RSA key pair (public & private)
  2. Public keys shared with all users
  3. Server generates random DES key per message
  4. DES key encrypted with recipient's RSA public
  5. Message encrypted with DES
  6. Recipient decrypts DES key with private RSA
  7. Recipient decrypts message with DES key
